# Usage Guide

## Getting Started

This guide covers the basic usage of Advanced Heliotrope.

## Creating the App Instance

1. Navigate to Apps in your Hubitat hub
2. Click "Add User App"
3. Select "Advanced Heliotrope"
4. Configure your location settings
5. Click "Done"

## Configuring Sky Regions

Sky regions define areas of the sky for triggering automations.

### Adding a Sky Region

1. Open the Advanced Heliotrope app
2. Select "Add Sky Region"
3. Configure the region parameters:
   - Name
   - Azimuth range (start and end angles)
   - Elevation range (start and end angles)
4. Save the region

## Using with Rule Machine

Advanced Heliotrope attributes can be used in Rule Machine for advanced automations.

## Troubleshooting

If you encounter issues, check the Hubitat logs for error messages.
